# TechZone: End-to-End Sales Intelligence Dashboard

## 📝 وصف المشروع (Project Overview)
**TechZone** هو مشروع تحليلي متكامل لمحاكاة بيانات سلسلة متاجر إلكترونيات متنوعة. الهدف من المشروع ليس فقط عرض البيانات، بل بناء "نظام معلوماتي" متكامل يبدأ من تصميم قاعدة البيانات وصولاً إلى لوحة التحكم التفاعلية التي تدعم اتخاذ القرار.

## ⚙️ الهندسة المعمارية للبيانات (Data Engineering)
تم بناء هذا المشروع من الصفر (Scratch) لمحاكاة بيئة عمل حقيقية:
1. **قاعدة البيانات (SQL Server):** قمت بإنشاء قاعدة بيانات عبر **SSMS** وتطوير سكربت SQL خاص لتوليد بيانات وهمية واقعية (100 عميل، 500 طلب، مع علاقات ربط قوية بين الجداول).
2. **منطق العمل (SQL Logic):** قمت بكتابة استعلامات SQL معقدة تمثل "العمود الفقري" للمشروع؛ حيث تُحدد هذه الاستعلامات المنطق الحسابي للأرباح، تقسيم العملاء (Segmentation)، وتقييم أداء الفروع.

## 📊 معالجة البيانات والتحليل (Data Transformation & Analytics)
بعد هندسة البيانات في SQL، انتقلت إلى **Power BI**:
* **Power Query:** لتنظيف البيانات، دمج الجداول، وهيكلة النموذج ليكون جاهزاً للتحليل.
* **DAX (Data Analysis Expressions):** تم بناء دوال DAX ديناميكية لإنشاء مقاييس (Measures) تفاعلية تتيح للمستخدم استكشاف الأداء بعمق.

## 💡 الرؤى الاستراتيجية (Business Insights)
* **تحليل الأداء التشغيلي:** مقارنة أداء الفروع بناءً على هوامش الربح وليس فقط حجم المبيعات.
* **تحليل سلوك العملاء:** تصنيف العملاء لتحديد الأكثر ولاءً والأكثر ربحية.
* **تحليل المنتجات:** ربط التكلفة بسعر البيع لتحديد "محركات الربح" الحقيقية في المتجر.
