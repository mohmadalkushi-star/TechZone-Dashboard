# TechZone: End-to-End Sales Intelligence Dashboard

## 📝 وصف المشروع (Project Overview)
**TechZone** هو مشروع تحليلي متكامل لمحاكاة بيانات سلسلة متاجر إلكترونيات. الهدف من المشروع ليس فقط عرض البيانات، بل بناء "نظام معلوماتي" متكامل يبدأ من قاعدة البيانات وصولاً إلى لوحة التحكم التفاعلية.

## ⚙️ الهندسة المعمارية للبيانات (Data Engineering)
هذا المشروع تم بناؤه من الصفر (Scratch):
1. **قاعدة البيانات (SQL Server):** قمت بإنشاء قاعدة بيانات عبر **SSMS** وتطوير سكربت SQL خاص لتوليد بيانات وهمية واقعية تشمل (100 عميل، 500 طلب، وعلاقات ربط بين الجداول).
2. **منطق العمل (SQL Logic):** قمت بكتابة استعلامات (SQL Queries) معقدة تمثل "العمود الفقري" للمشروع؛ حيث تُحدد هذه الاستعلامات المنطق الحسابي للأرباح، تقسيم العملاء (Segmentation)، وتقييم أداء الفروع. 
   - *يمكنك الاطلاع على نموذج من منطق الاستعلامات في `image_e05cdd.png`.*

## 📊 معالجة البيانات والتحليل (Data Transformation & Analytics)
بعد بناء المنطق في SQL، انتقلت إلى **Power BI**:
* **Power Query:** تم استخدامه لتنظيف البيانات، دمج الجداول، وهيكلة البيانات لتكون جاهزة للتحليل.
* **DAX (Data Analysis Expressions):** تم بناء دوال DAX ديناميكية لإنشاء مقاييس (Measures) تفاعلية تمكن المستخدم من استكشاف الأداء بعمق.

## 💡 الرؤى الاستراتيجية (Business Insights)
* **تحليل الأداء التشغيلي:** مقارنة أداء الفروع بناءً على هوامش الربح وليس فقط حجم المبيعات.
* **تحليل سلوك العملاء:** تصنيف العملاء (Customer Segmentation) لتحديد الأكثر ولاءً والأكثر ربحية.
* **تحليل المنتجات:** ربط التكلفة بسعر البيع لتحديد "محركات الربح" الحقيقية في المتجر.
